###跨文档消息通信
####postMessage对象
        接收消息的窗口对象postMessage()
        一参 : 发送的数据，二参 : 发送的域
        交互方式:
            iframe：父页面 : contentWindow、子页面 : window.top
            窗口页: 父页面 : window.open、子页面 : window.opener
        接收事件：
            message
            ev.origin : 发送数据来源的域
            ev.data : 发送的数据
